# 📊 Consultar JSON em C#

![Consultar](https://github.com/user-attachments/assets/4cd22501-82a5-4c36-9786-492f471dd1fa)


Um programa simples em C# para consumir e exibir dados de uma API em formato JSON. Com uma interface intuitiva pelo console, o usuário pode listar todos os dados ou buscar informações específicas pelo ID. Ideal para quem deseja entender como integrar APIs e manipular dados JSON em C#.

---

## 🚀 Funcionalidades

🔎 **Listar JSON:** Consulta e exibe todos os dados da API.

🔢 **Buscar por ID:** Permite buscar e exibir um item específico pelo ID.

🖥️ **Interface Simples:** Navegação fácil por meio de um menu no console.

🌐 **Consumo de API:** Integração com a API [JSONPlaceholder](https://jsonplaceholder.typicode.com/).

---

## 🛠️ Como Executar o Projeto

1. **Clone o repositório:**

```bash
git clone https://github.com/seuusuario/ConsultarJson.git
```

2. **Abra o projeto no Visual Studio ou editor de sua preferência.**

3. **Compile e execute o projeto:**

```bash
dotnet run
```

4. **Utilize o menu para navegar entre as opções:**

- **1:** Listar todos os dados JSON
- **2:** Buscar por ID
- **3:** Sair
